Describe Azure Compute Services.
Azure compute is an on-demand computing service for running [[+cloud_based_applications]] .	[[https://docs.microsoft.com/en-us/learn/modules/azure-compute-fundamentals/overview|[ref 1]]]

How do you pay for Azure compute services?
You pay only for the resources you use, and only for as long as you're using them.	[[https://docs.microsoft.com/en-us/learn/modules/azure-compute-fundamentals/overview|[ref 1]]]

How are Azure compute resourses available?
On demand.	[[https://docs.microsoft.com/en-us/learn/modules/azure-compute-fundamentals/overview|[ref 1]]]

How soon can Azure compute resources be available?
in minutes or even seconds.	[[https://docs.microsoft.com/en-us/learn/modules/azure-compute-fundamentals/overview|[ref 1]]]

What does azure support?	[[https://docs.microsoft.com/en-us/learn/modules/azure-compute-fundamentals/overview|[ref 1]]]
* development
* testing
* running applications
* extending your datacenter
