====== NuGet ======
Created Wednesday 25 August 2021

Source documents:
https://en.wikipedia.org/wiki/NuGet

What is NuGet?
Nuget is a package manager designed to enable developers to share reusable code.

Describe a NuGet package.
* a single file
* bears a .nupack or .nupkg filename extension
* contains .NET assemblies and their needed files
