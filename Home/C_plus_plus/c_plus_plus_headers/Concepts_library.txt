What is a C++ concept?
Concepts are an extension to the [[+templates]] feature provided by the C++ programming language.	[[https://en.wikipedia.org/wiki/Concepts_(C%2B%2B)|[ref 2]]]

What are th 4 main uses of concepts in C++?	[[https://en.wikipedia.org/wiki/Concepts_(C%2B%2B)|[ref 2]]]
1. introducing type-checking to template programming.
2. simplified compiler diagnostics for failed template instantiation.
3. selecting function template overloads and class template specializations based on type properties.
4. constraining automatic type deduction.
