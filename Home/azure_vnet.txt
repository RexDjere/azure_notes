What is an Azure Virtual Network (VNet)?
Azure Virtual Network is the fundamental building block for your private network in Azure.	[[https://docs.microsoft.com/en-us/azure/virtual-network/virtual-networks-overview|[ref 1]]]

What does Azure VNet enable?
Vnet enables many types of Azure resources, such as Azure Virtual Machines (VM),
to securely communicate with each other, the Internet, and on-premises networks.
