What is Amazon Athena?	[[https://docs.aws.amazon.com/whitepapers/latest/aws-overview/analytics.html|[ref 1]]]
* an interactive query service
* makes it easy to analyze data in Amazon S3 using standard SQL
* Athena is serverless, so there is no infrastructure to manage, and you only pay for the queries that you run
